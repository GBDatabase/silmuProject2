	(1) IOC(제어의 역전)

		  직접 생성하거나 제어하는것이 X
		- 코드에서 객체를 생성하지 않고 선언 후 받아온 객체를 B에 할당
	(2) DI (dependancy Injection)
		- 의존성 주입 (=의존 객체 조립)
		``` java
			public class A { 
			@aurowired
			B b;
			}
		```
